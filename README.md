## 用python一步一步搭建高并发rpc服务

#### 开发环境
```
系统: MacOSX 10.14.4
语言: python 3.7.2
```

#### RPC服务器-研究分7个阶段循序渐进
1. 单线程同步
```
由单进程-单线程的方式服务
```
2. 多线程同步
```
由主进程-开启多线程方式服务
```
3. 多进程同步
```
主进程收到客户端链接后fork出子进程为客户端提供服务
```
4. PreForking多进程同步
```
预先fork出多个进程共同监听套接字提供服务
```
5. 单进程异步
```
非阻塞IO/事件监听,单进程异步服务
```
6. PreForking多进程异步
```
预先fork出多个进程,每个进程内都采用异步的方式提供服务
```
7. 多进程传递文件描述符(同步)
```
采用Node cluster多进程模型,由master进程传递fd给slave进程处理. 避免了多个salve竞争处理导致的`马太效应`
```
